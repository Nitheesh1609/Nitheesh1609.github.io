CPU Profiler

The goal of this project is to design a CPU profiling tool. The tool will be designed as a single kernel module, which when loaded, keeps track of statistics each task on the system, like total time spent on the CPU and the number of times a task was scheduled in/out. The module will display your profiling results using the proc file system.

Linux Kernel DataStructures

The goal of this project is to get used to essential data structures in Linux Kernel. Basically, to develop a simple kernel module manipulating a module parameter and the kernel dataStructures. More specifically, the kernel module takes a comma-separated string upon its loading and it parses the given string and adds each integer numbers to the kernel dataStructures For ease of testing, we will add a file in /proc file system to print out the contents of data structures.
